# Nexie  - AI 编程智能体
Nexie(奈西,读音:/'neksi/,,,,中文昵称:力袭,寓意:努力逆袭)
## 🖥️ 桌面端
文件：Nexie.zip

功能：
- AI 驱动代码生成与修改
- Git 全覆盖（19个命令）status / add / commit / push / pull / branch / merge / rebase /cherry-pick / tag / clone / remote / init / create_pr自动生成 commit message
-  多数据库 SQLite / MySQL / PostgreSQL / MongoDB 查询、表结构、导出
-  网络能力 GET/POST/PUT/DELETE/PATCH + 网页抓取 + 文件下载 + 自定义头
-  包管理 npm / yarn / pnpm / pip / cargo / go mod 自动检测与操作
- 智能文件管理
- 桌面自动化操作
- 命令行一键执行
- 代码质量审查

安装：
1. 下载 Nexie.zip
2. 解压到任意文件夹
3. 双击运行 Nexie.exe

系统要求：Windows 10/11，8GB+ 内存
