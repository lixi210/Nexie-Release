# Nexie - AI 编程智能体

Nexie 是一个强大的 AI 编程智能体，支持 Windows 桌面端和 Android 手机端。

## 📥 下载

| 版本 | 文件 | 说明 |
|------|------|------|
| 🖥️ 桌面端 | `Nexie.zip` | Windows 10/11 安装包 |
| 📱 手机端 | `Nexie-mobile.zip` | Android 安装包 |

从 [Releases](https://github.com/lixi210/Nexie-Release/releases) 页面下载。

## 🖥️ 桌面端使用说明

1. 下载 `Nexie.zip`
2. 解压到任意目录
3. 运行 `Nexie.exe`

## 📱 手机端使用说明

1. 下载 `Nexie-mobile.zip`
2. 解压后在手机上安装 APK

## ⚙️ 系统要求

- 桌面端：Windows 10 / 11，建议 8GB 以上内存
- 手机端：Android 8.0 以上
